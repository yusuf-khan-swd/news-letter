var
  • Using var to declare a variable can be re-assign. Example :
    • var a = 10; a = 20
  • Using var to declare a variable can be re-declare. Example :
    • var a = 10; var a = 20
  • Using var to declare a variable make that variable is GLOBAL variable.
let
  • Using let to declare a variable can be re-assign. Example :
    • let a = 10; a = 20
  • Using let to declare a variable can't be re-declare. Example :
    • let a = 10; let a = 20; This will give Error
  • Using let to declare a variable make that variable is SCOPE variable.
const
  • We can't leave without assigning a value to const variable
    • const a; a = 20 This will giveError
  • Using const to declare a variable can't be re-assign. Example:
    • const a = 10; a = 20; This will give Error
  • Using const to declare a variable can't be re-declare.Example:
    • const a = 10; const a = 20; This will give Error
  • Using const to declare a variable make that variable is SCOPE variable.

Regular Function
  • Regular function declare with keyword function
  • Regular function can be function expression. like :
    • let myFunc = function() {}
  • Regular function can be named or anonymous function. like:
    • let myFunc = function(input) { return input + 2 }; let myFunc2 = named function(input1, input2) { return input1 + input2 } myFunc3 function(a, b) { return a + b }
Arrow Function
  • Arrow function declare with fat arrow =>
  • Arrow function does not have function keyword
  • Its look like this : let show = input => input + 20
    • Its example of one parameter input.
  • In case of no parameter input we have to use parenthesis. let show = () => 90 + 20
  • If we have multiple input and our code can be multiline we have use curly bracket and explicitly return value if we needed. let sumOf = (input1, input2) => { let sum1 = input1 + 5; let sum2 = input2 + 10; return (sum1 + sum2) % 2 === 0}

map
  • loop throw the every array elements
  • and do given task on every single element
  • and return a new array with completed task
forEach
  • loop throw the every array elements
  • and do given task ove every single element
  • forEach not return a new array
filter
  • loop throw the every array elements
  • takes condition of true or false
  • and given condition filter the element that fulfill the condition
  • and return new array of that filtered elements
find
  • loop throw the every array elements
  • reduce the array to single value of given condition
  • only return the final result
